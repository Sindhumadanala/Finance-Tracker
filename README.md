## Description

Finance Tracker is a simple and responsive web application that helps users manage their daily income and expenses. Users can add transactions by entering the date, amount, and transaction type. The application automatically calculates and displays Total Income, Total Expense, and Balance in real time. It also allows users to delete transactions while updating the financial summary instantly. The interface is designed to work smoothly on both desktop and mobile devices.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap 5
* Google Fonts (Chelsea Market)

## Key Features

* Add Income and Expense transactions
* Automatic calculation of Income, Expense, and Balance
* Dynamic transaction table generation
* Delete transactions with instant recalculation
* Responsive design for Desktop and Mobile devices
* Attractive and user-friendly interface
* Custom icons and modern card-based dashboard
